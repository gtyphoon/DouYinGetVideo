### 2、注意事项
#### ▶环境准备
基于驱动chrome完成下载，需要将chrome浏览器及其驱动放入到文件夹chrome。\
驱动官方下载地址：https://googlechromelabs.github.io/chrome-for-testing/

#### ▶不足之处
- ##### ⨉ 人机验证
    由于抖音平台会在网页刚打开过程中跳出验证码进行人机验证，该部分需要人工手动介入。
- ##### ⨉ 主页视频下载尚未完善
- ##### ⨉ 下载的视频命名问题尚未完善
    当前为将下载的视频文件名（ID数字）与视频名称及视频链接放在当前目录中的CSV文件中
